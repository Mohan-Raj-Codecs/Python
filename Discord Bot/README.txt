Purpose:
# This is Just a Simple Bot For Managing your Bs Server from your Discord Server Commands

Configs:
# There are some to do before you Execute Script

Steps :

Step 1 : Open ssh.py
Step 2 : Change the ip and user
Step 3 : If you are using Password For linux Server Uncomment the passwd variable and Line 17
Step 4 : If you are using PEM file to Authorize to Linux Server then change the pem_file to the name of your Pem file
Step 5 : Copy and Paste Your PEM file in the Same Directory where Scripts are Loaded
Step 6 : Open the bot.py
Step 7 : Fill the god list with your Discord ID (Something Like this 574188245206958094) ,You can have Full Access to Commands
Step 8 : Now Done Load Scripts in Server and Launch it 

INFO'S :
1. Server Has Three Roles in it : god,Owners,Admins
2. Admin has Access to start and stop server
3. Owners has Access to add Admins and Owners , and has Access to Server
4. god can Remove Roles from user's and have FULL ACCESS
