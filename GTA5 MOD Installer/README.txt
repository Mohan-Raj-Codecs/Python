1#Copy and paste the GTA V.exe Game Folder Path in Prompt and Press Enter

2#Done It will Copy all the Menyoo MODS to GTAV, Ready to Run

3#You can see a File(MOD Uninstaller.bat) in GTA V path , Double Click on that to Uninstall the MODS
